# Vim ťahák (SK)

Moderný HTML ťahák pre Vim v slovenčine. Obsahuje prehľad základných príkazov, rýchlu navigáciu cez kotvy a tlačovú verziu optimalizovanú pre PDF/print.

## Štruktúra
- `vim-tahak.html` – hlavná stránka s obsahom aj štýlmi.
- `Vimlogo.svg` – logo použité v hlavičke.
- `README.md` – tento popis.

## Poznámky k úpravám
- Stránka je samostatný HTML súbor bez build krokov. Úpravy stačí spraviť priamo v `vim-tahak.html`.
- Pri úpravách rozloženia sleduj grid triedy `tall-card` a `full-card`, ktoré ovplyvňujú desktop aj tlač.
